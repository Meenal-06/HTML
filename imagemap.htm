Image Map

The HTML tag defines an image map. An image map is an image with clickable areas. tags.

Types of image map

There are two types of image maps
1. server side
2. client side.

server side

The is map attribute is a boolean attribute. When present, it specifies that the image is part of a server-side image map.

client side

Client-side imagemaps are clickable images that don't require a CGI program to make them work. ... Because they are faster and more reliable, client-side imagemaps have largely supplanted server-side imagemaps
